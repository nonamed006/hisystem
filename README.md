## 프로젝트 소개

**1. 개발기간**<br>
<p>2021.11~2021.12 (4주) </p>

**2. 프로젝트 개요**<br>
<p>- 접수, 진료, 수납으로 진행되는 병원 프로세스 이해 및 구현</p>
<p>- 채팅, 칸반보드, To-Do List, 근무표 등 편의 서비스 구현 </p>
<p>- Spring Boot를 활용한 JPA(ORM) 이해와 활용</p>
<p>- React로 SPA & Client Side Rendering 이해와 활용</p>
